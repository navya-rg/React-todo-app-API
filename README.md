# React-todo-app-API
Node.js APIs for React-todo-app (https://github.com/navya-rg/React-todo-app)

### APIs
1. **List management**
    1. Get all lists
    2. Get list (id)
    3. New list
    4. Update list
    5. Star list
    6. Delete list

1. **Task management**
    1. Get all tasks
    2. Get tasks (listId)
    3. New task
    4. Update task
    5. Complete task
    6. Delete task

1. **Database management**
    1. Recreate all sqlite tables
